# API Key 配置

## 以 DeepSeek 为例，拿 Key 的流程：

1. 在 AI 填空助手里选择 DeepSeek，点击旁边的“API 文档”。

2. 文档页面里找到“apply for an API key”之类的入口，点进去。

3. 进入控制台后创建 API key。名称只是备注，方便你自己认，真正有用的是创建出来的那一长串
乱七八糟的以 sk- 开头的字符串。

4. 这个 key 往往只会完整显示一次！复制后自己保管好，不然丢了就得重新创建，泄露了就可能
被别人拿去刷爆你的余额！

5. 如果测试连接失败，优先检查四件事：Key 对没对、模型 ID 对没对、服务商选没选错、账户里
还有没有余额。
