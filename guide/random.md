# 随机 IP 与随机 UA

此处设置项的重要程度等同于 [运行参数](./runtime.md)，需要仔细阅读！

![随机IP和随机UA设置](../assets/random_1.png)

## 随机 IP

几乎是使用频率最高、需求最强的设置项，它负责让访问看起来来自不同地区，尤其应用于抢志愿、抢选课或其他特殊需求场景。

> 本程序内置了公开服务的商用代理源，因此存在付费额度限制。

:::warning ⚠️警告
针对问卷星问卷，**必须启用随机 IP** 以应对智能验证，否则将极大拖慢作答效率

腾讯问卷与见数无此限制
:::

### 自定义代理源

本程序支持接入用户自备的代理池，此时可不受商用代理源的额度限制。

连接质量取决于用户自备代理池的质量，使用前请务必测试连接，确保本机可通过自身代理源访问鉴权（如白名单、账密等）。

- 网络中公开免费的代理池质量参差不齐，绝大多数免费ip甚至根本**加载不出问卷**，使用前请务必测试连接。

> 国内地区的代理商要求实名认证，使用时请遵守代理提供商的服务条款。品赞已经提供了使用教程及协议：https://www.ipzan.com/help
>
> 提供的免费试用的额度大约能够支持 2500 次填写。

:::warning ⚠️警告
仅支持 HTTP(s) 协议和 json 返回格式的代理提取 API
:::

如果你根本就不知道什么是自定义代理源、什么是 API 地址，请保持默认代理源。

## 随机 UA

负责模拟不同来源环境，如微信、手机浏览器、链接访问

支持自定义不同 UA 的提交占比，不启用时默认为 100% 直链访问

如果你只是普通跑问卷，或者说无人在意你的问卷是哪来的，这个功能可以不启用，保持默认即可。针对抢志愿、抢选课等特殊需求场景，微信无脑拉满 100% 即可。
