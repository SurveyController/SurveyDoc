# 下载与安装

本页演示了SurveyController下载与安装的具体详细操作流程，以及一些常见问题的解决方法

## 安装教程

双击下载到的安装包，按提示安装。

![安装](assets/install_1.png)

大多数人保持默认一路下一步就行。(如果安装向导询问安装位置或开始菜单文件夹，按你自己的习惯选；不想折腾就继续默认。)

## 启动程序

安装完成后，默认勾选运行SurveyController，即可进入主界面。

![启动程序](assets/install_2.png)

## 首次启动问题检查

安装完成后桌面一般会出现快捷方式。要是没看到，先检查一下你那乱得像灾后废墟的桌面。

第一次启动时窗口可能会慢个三四秒才会出现窗口，这是正常现象。

如果程序打不开，请保留完整报错截图和文字信息，再去群里或 GitHub 反馈。只说一句“打不开”基本等于没说。
