# Markdown 扩展示例

本页演示了一些内置 Markdown 扩展能力。

## 语法高亮

VitePress 使用 [Shiki](https://github.com/shikijs/shiki) 提供，并支持行高亮等额外功能：

## 自定义容器

**输入**

```md
::: info
这是一个信息提示框。
:::

::: tip
这是一个提示框。
:::

::: warning
这是一个警告框。
:::

::: danger
这是一个危险警告框。
:::

::: details
这是一个详情折叠块。
:::
```

**输出**

::: info
这是一个信息提示框。
:::

::: tip
这是一个提示框。
:::

::: warning
这是一个警告框。
:::

::: danger
这是一个危险警告框。
:::

::: details
这是一个详情折叠块。
:::
